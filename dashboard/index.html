Stream Keys

Securely manage your YouTube and Facebook stream keys

Security Notice — Stream keys are stored in /opt/streaming-backend/keys/stream_keys.env with chmod 600. Never commit this file to git or share it. Rotate keys after every event.
YouTube Stream Key
How to get: YouTube Studio → Go Live → Stream → Copy Stream Key
Update in rtmp.conf:
sudo nano /etc/nginx/rtmp.conf
sudo nginx -s reload
Get Key from YouTube Studio ↗
Facebook Stream Key
How to get: Facebook → Professional Dashboard → Live Video → Use Stream Key
Update in stream_keys.env:
sudo nano /opt/streaming-backend/keys/stream_keys.env
sudo systemctl restart fb-relay

ℹ️ How It Works

1️⃣
OBS Streams to Relay
OBS sends RTMP stream to Nginx relay on port 1935 as usual.
2️⃣
Nginx Creates HLS Chunks
Nginx splits stream into 3-second .ts video chunks + .m3u8 playlist automatically.
3️⃣
Browser Plays via HLS.js
Viewer opens link → HLS.js downloads chunks → seamless live video in any browser.
4️⃣
Share the Link
Post on WhatsApp group — anyone clicks & watches live. ~5 sec delay. No account needed.
